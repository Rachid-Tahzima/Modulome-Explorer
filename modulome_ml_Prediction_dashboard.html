The RNA Virus Modulome - ML Prediction Dashboard

Ensemble random-forest models predicting virus family, genus, transmission mode and vector traits from proteome and modular architecture.

1. Tasks & Targets

  • Family_NCBI (multiclass)  from proteome_length, perc_prot_disorder, Plant_Virus_ID encoding and all SSF / G3DSA / PF modules.
  • Genus_NCBI (multiclass)  same features plus Family_NCBI encoding.
  • TransmissMode (multiclass)  molecular features (no explicit taxon encoding in this version).
  • 20 vector traits (multilabel)  Sap_transmissible, Aphids, Seed, Soil, etc. via multi-output forests.

5. Modelling notes

  • Random Forest ensembles act as fast, accurate baselines on this modular feature space.
  • Multi-output forests jointly model the 20 vector traits, capturing shared structure between, e.g., aphid- and seed-borne viruses.
  • Metrics include accuracy, macro-F1, weighted-F1, precision, recall and MCC; AUROC / PR-AUC can be added per trait for more detailed evaluation.
